## ‚ú® Features

* **Beautiful GUI:** A modern, dark-themed, and user-friendly interface built with Python's Tkinter library.
* **Unbeatable AI:** The AI opponent is powered by the **Minimax algorithm**, a classic game theory algorithm that finds the optimal move in any situation. It will always win or force a draw.
* **Dynamic UI:** The app includes a dynamic turn indicator to show whose move it is.
* **Polished Feel:** Features energetic pop-up messages, custom fonts, and a slight delay on the AI's move to feel more natural.
* **Standalone:** No external libraries are needed! It runs on standard Python.
